Elementos Semánticos y de Sección en HTML

Etiqueta Función
<header> Define un encabezado para un documento o sección. Normalmente contiene el logo, menú o título.
<footer> Define el pie de página para un documento o sección. Suele contener información de contacto o enlaces.
<nav> Define un contenedor para enlaces de navegación, como menús.
<main> Define el contenido principal del documento, excluyendo encabezados, pies de página y barras laterales.
<section> Define una sección temática en un documento. Agrupa contenido relacionado.
<article> Define un contenido independiente como una publicación de blog, artículo o comentario.
<aside> Define contenido secundario, como barras laterales, anuncios o enlaces relacionados.
<figure> Define contenido multimedia, como imágenes o diagramas, junto con un subtítulo.
<figcaption> Agrega un subtítulo o descripción a un elemento <figure>.
<mark> Resalta texto para llamar la atención o marcarlo como relevante.
<time> Define una fecha o una hora legible por máquina.
<details> Define un elemento desplegable que muestra u oculta contenido adicional.
<summary> Proporciona un encabezado visible para un elemento <details>.
<address> Define información de contacto, como direcciones o correos electrónicos.
