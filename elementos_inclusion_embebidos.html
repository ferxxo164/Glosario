Elementos de Inclusión y Embebidos en HTML

Etiqueta Función
<iframe> Permite incrustar otra página web dentro del documento actual.
<embed> Incrusta contenido multimedia o aplicaciones externas, como archivos PDF o Flash.
<object> Incorpora objetos multimedia, como videos, sonidos o documentos.
<param> Define parámetros para un elemento <object>.
<video> Permite la inserción y reproducción de videos en una página web.
<audio> Permite la inserción y reproducción de archivos de audio en una página web.
<source> Define múltiples recursos multimedia para los elementos <video> y <audio>.
<track> Proporciona pistas de texto, como subtítulos o descripciones, para <video> y <audio>.
<picture> Define múltiples versiones de una imagen para diferentes tamaños de pantalla o resoluciones.
<img> Inserta imágenes en una página web.
<map> Define un mapa de imágenes con áreas clickeables.
<area> Define una región dentro de un <map>, que actúa como enlace.
