Elementos Interactivos y Gráficos en HTML

Etiqueta Función
<button> Define un botón interactivo que puede ejecutar acciones cuando se presiona.
<input> Permite la entrada de datos en formularios, como texto, contraseñas y más.
<select> Define un menú desplegable para seleccionar opciones.
<option> Define una opción dentro de un elemento <select>.
<textarea> Permite la entrada de texto en varias líneas dentro de un formulario.
<label> Define una etiqueta para un elemento de formulario, mejorando la accesibilidad.
<fieldset> Agrupa elementos de un formulario bajo una misma categoría.
<legend> Define un título o descripción para un <fieldset>.
<canvas> Permite dibujar gráficos mediante scripts como JavaScript.
<svg> Define gráficos vectoriales escalables mediante XML.
<video> Permite la inserción y reproducción de videos en una página web.
<audio> Permite la inserción y reproducción de audio en una página web.
<progress> Representa visualmente el progreso de una tarea.
<meter> Representa visualmente una medida dentro de un rango conocido.
<details> Crea un elemento interactivo desplegable para mostrar u ocultar contenido.
<summary> Define el encabezado visible para un elemento <details>.
