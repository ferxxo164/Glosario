Etiquetas HTML y su Función

Etiqueta Función
<div> Define un contenedor genérico en bloque para agrupar elementos y aplicar estilos o scripts.
<section> Define una sección independiente dentro de un documento, útil para estructurar contenido temáticamente.
<header> Representa un encabezado para un documento o sección, generalmente contiene logotipos, menús o títulos.
<footer> Define el pie de página de un documento o sección, incluye información como enlaces, derechos de autor, etc.
