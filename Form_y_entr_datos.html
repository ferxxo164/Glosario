Formularios y Entradas de Datos en HTML

Etiqueta Función
<form> Define un formulario para recopilar datos del usuario. Usa los atributos action (destino) y method (GET o POST).
<input> Define un campo de entrada para datos. Acepta diferentes tipos como texto, contraseña, número, correo, etc.
<label> Asocia un texto descriptivo con un elemento de formulario, mejorando la accesibilidad.
<textarea> Define un área de texto para ingresar múltiples líneas de contenido.
<select> Define un menú desplegable (lista de opciones).
<option> Define una opción dentro de un menú desplegable <select>.
<button> Define un botón para enviar o restablecer un formulario, o ejecutar una acción específica.
<fieldset> Agrupa elementos relacionados dentro de un formulario.
<legend> Agrega un título descriptivo a un grupo de elementos dentro de un <fieldset>.
<datalist> Define una lista de opciones predefinidas para un campo de entrada.
<output> Muestra el resultado de un cálculo o acción en un formulario.
<progress> Representa el progreso de una tarea, como una barra de carga.
<meter> Representa un valor dentro de un rango conocido, como un medidor.
