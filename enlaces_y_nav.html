Enlaces y Navegación en HTML

Etiqueta Función
<a> Define un enlace a otra página o recurso. Usa el atributo href para especificar la URL de destino.
<nav> Define un contenedor para enlaces de navegación, como menús o barras de navegación.
<link> Usada para enlazar archivos externos, como hojas de estilo (CSS) o íconos de la página.
<area> Define un área activa dentro de un mapa de imágenes que se convierte en un enlace.
<map> Define un mapa de imágenes para crear áreas con enlaces dentro de una imagen.
<iframe> Inserta un marco en línea que muestra otra página web dentro de la página actual.
<base> Define la URL base para todos los enlaces relativos en un documento.
