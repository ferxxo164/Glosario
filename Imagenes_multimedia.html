Imágenes y Multimedia en HTML

Etiqueta Función
<img> Inserta una imagen en la página. Usa el atributo src para la ruta de la imagen y alt para la descripción alternativa.
<picture> Define un contenedor para mostrar diferentes versiones de una imagen según las condiciones del dispositivo (como tamaño o resolución).
<source> Especifica recursos multimedia alternativos dentro de <picture> o <video>.
<figure> Define contenido independiente (como una imagen, diagrama o gráfico) con la posibilidad de incluir un subtítulo.
<figcaption> Agrega un subtítulo o descripción a un elemento <figure>.
<audio> Inserta un reproductor de audio para archivos de sonido. Acepta formatos como MP3, OGG o WAV.
<video> Inserta un reproductor de video. Acepta formatos como MP4, WebM y Ogg.
<track> Agrega pistas de texto (como subtítulos o descripciones) a un elemento <video> o <audio>.
<embed> Incorpora contenido multimedia externo, como videos o aplicaciones interactivas.
<object> Incrusta objetos multimedia como imágenes, videos o documentos PDF.
