Etiquetas Obsoletas de HTML

Etiqueta Función
<acronym> Definía acrónimos, reemplazado por <abbr>.
<applet> Incorporaba applets de Java, reemplazado por <object>.
<basefont> Definía la fuente base para todo el documento, reemplazado por CSS.
<big> Aumentaba el tamaño del texto, reemplazado por CSS.
<center> Centraba contenido, reemplazado por CSS.
<font> Definía el tipo, color y tamaño de fuente, reemplazado por CSS.
<frame> Definía marcos en una página, reemplazado por <iframe>.
<frameset> Agrupaba marcos, reemplazado por CSS y <iframe>.
<isindex> Proporcionaba una caja de búsqueda, reemplazado por <input>.
<marquee> Generaba texto en movimiento, reemplazado por CSS y JavaScript.
<noframes> Proporcionaba contenido alternativo para navegadores sin soporte de marcos, ahora innecesario.
<s> Tachaba texto, reemplazado por <del> o CSS.
<strike> Tachaba texto, reemplazado por <del> o CSS.
<tt> Aplicaba una fuente monoespaciada, reemplazado por CSS.
<u> Subrayaba texto, reemplazado por CSS.
