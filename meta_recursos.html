Meta-información y Recursos en HTML

Etiqueta Función
<meta> Proporciona metadatos sobre el documento, como la codificación y descripción.
<title> Define el título del documento, que aparece en la pestaña del navegador.
<base> Especifica la URL base para todos los enlaces relativos en el documento.
<link> Enlaza recursos externos, como hojas de estilo CSS.
<style> Define reglas de estilo CSS dentro del documento HTML.
<script> Incorpora o enlaza scripts de JavaScript en el documento.
<noscript> Proporciona contenido alternativo para navegadores sin soporte de JavaScript.
<html> Elemento raíz de un documento HTML, puede incluir atributos como el idioma.
<head> Contiene metadatos, enlaces a recursos externos y el título del documento.
<body> Contiene el contenido visible del documento HTML.
